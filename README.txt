Cuando se ejecuta el programa, se ve el Menu Principal:

Bienvenido al sistema de gestion
¿Que desea gestionar?:
1. Disciplinas
2. Socios
3. Pagos
4. Inscripciones
0. Salir

Elegir una opcion escribiendo el numero correspondiente:

DICIPLINAS:
Permite administrar las disciplinas deportivas del club.

1. Agregar disciplina: Ingresar nombre, tipo (EQUIPO o INDIVIDUAL) y capacidad maxima.

2. Eliminar disciplina: Eliminar una disciplina ingresando su ID.

3. Buscar disciplina: Buscar por nombre, ID o tipo.

4. Mostrar disciplinas: Lista todas las disciplinas registradas.

SOCIOS
Permite administrar los socios del club.

1. Agregar socio: Registrar un nuevo socio ingresando nombre, edad y DNI.

2. Eliminar socio: Eliminar un socio indicando su ID o DNI.

3. Buscar socio: Consultar un socio especifico por su nombre, DNI o ID.

4. Mostrar todos los socios: Lista completa de socios registrados en el club.

5. Mostrar socios morosos: Mostrar todos los socios que tienen deudas.

0. Volver: Retorna al menu principal.

PAGOS
Gestiona el registro y consulta de pagos de socios:

1. Registrar nuevo pago: Elegir un socio y una disciplina, e ingresas el importe y concepto.

2. Consultar pagos de un socio: MostraR todos los pagos hechos por un socio especifico.

3. Consultar recaudacion por disciplina: Ver cuanto dinero recaudo una disciplina.

4. Mostrar todos los pagos: Mostrar el listado completo de todos los pagos registrados.

INSCRIPCIONES
Gestiona las inscripciones de los socios a disciplinas:

1. Inscribir socio a disciplina: Elegir un socio y una disciplina e incribirlo.

2. Eliminar inscripcion de socio: Cancelar la inscripcion de un socio en una disciplina.

3. Ver socios inscriptos en una disciplina: Mostrar todos los socios anotados en una disciplina.

0. Salir
Finaliza la ejecucion del programa.
